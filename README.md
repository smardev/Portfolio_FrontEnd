```bash
git clone https://github.com/smardev/Portfolio_FrontEnd.git
```

2. Navigate to the project directory:
```bash
cd your-project-directory
```

3. Install the dependencies:
```bash
npm install
```

4. Serve the application:
```bash
ng serve
```
